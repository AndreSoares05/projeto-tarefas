# 📋 Sistema de Tarefas

## 📖 Descrição do Sistema

O Sistema de Tarefas é uma aplicação Full Stack desenvolvida para gerenciamento de tarefas de forma simples e prática.

O sistema permite:

- Cadastrar tarefas
- Listar tarefas
- Editar tarefas
- Excluir tarefas

Os dados são armazenados em banco de dados utilizando o Supabase e toda a comunicação entre o Front-end e o Back-end é feita através de uma API REST utilizando Node.js e Express.

---

# 🛠️ Tecnologias Utilizadas

## Back-end

- Node.js
- Express
- Supabase
- dotenv
- cors

## Front-end

- HTML5
- CSS3
- JavaScript

## Banco de Dados

- Supabase PostgreSQL
